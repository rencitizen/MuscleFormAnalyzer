# BodyScale 次期機能開発プロンプト

## 🎯 開発目標
**RIZAPを超えるAI搭載トレーニングアプリの完成**
既存の優秀なシステムに、データ可視化とアカウント機能を追加してユーザー体験を完璧にする

## 📊 現在完成している基盤システム
```
✅ AIフォーム分析（MediaPipe + 個人体格データ活用）
✅ 身体寸法測定（腕・脚の長さ等）
✅ トレーニング記録システム（100種目以上、部位別検索）
✅ データ永続化（PostgreSQL）
✅ 重量・回数・セット数記録
```

## 🚀 実装する新機能

### 1. データ可視化・分析機能（最優先）

**進捗グラフ機能**
```
- 種目別最大重量推移グラフ（線グラフ）
- トレーニングボリューム推移（重量×回数×セット）
- 週次/月次集計データ
- 個人記録（PR）ハイライト表示
- 前回比較機能
```

**カレンダー表示**
```
- 月間カレンダーでトレーニング実施日を色分け
- 日付クリックで詳細表示
- 連続トレーニング日数の表示
- 休息日の可視化
```

**統計ダッシュボード**
```
- 今月の合計トレーニング時間
- 最も頻繁に行う種目TOP5
- 部位別トレーニング頻度
- 成長率（前月比）
```

**データ分析機能**
```
- 個人記録一覧（種目別最大重量）
- 成長傾向分析
- パフォーマンス予測
- 目標達成率
```

### 2. ユーザーアカウント強化

**メール認証システム**
```
- ユーザー登録（メール + パスワード）
- ログイン・ログアウト機能
- パスワードリセット機能
- メール認証による本人確認
```

**データ管理機能**
```
- 全データのバックアップ・復元
- アカウント削除機能
- データエクスポート（CSV/JSON）
- プライバシー設定
```

**セキュリティ強化**
```
- セッション管理の改善
- パスワード暗号化
- 不正アクセス防止
- ログイン履歴
```
