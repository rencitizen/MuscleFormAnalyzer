# BodyScale トレーニング記録機能 拡張プロンプト

## 🎯 追加機能要件

### 1. トレーニング種目データベース構築

**部位別種目分類：**

**脚部（Legs）**
- **大腿四頭筋**: スクワット、レッグプレス、レッグエクステンション
- **ハムストリングス**: ルーマニアンデッドリフト、RDL、レッグカール、グッドモーニング
- **お尻（臀筋）**: ヒップスラスト、ブルガリアンスクワット、ケーブルキックバック、アブダクション、アダクション
- **ふくらはぎ**: スタンディングカーフレイズ、シーテッドカーフレイズ

**胸部（Chest）**
- **大胸筋**: ベンチプレス、ダンベルプレス、インクラインベンチプレス、インクラインダンベルプレス、デクラインベンチプレス、デクラインダンベルプレス、ディップス、チェストフライ、プッシュアップ

**背中（Back）**
- **広背筋**: ラットプルダウン、チンニング、懸垂、ストレートアームプルダウン
- **僧帽筋・脊柱起立筋**: デッドリフト、バーベルロウ、ダンベルロウ、シーテッドロウ、ハイパーエクステンション、シュラッグ

**肩（Shoulders）**
- **三角筋**: ショルダープレス、サイドレイズ、フロントレイズ、リアデルトフライ、アップライトロウ

**腕（Arms）**
- **上腕二頭筋**: バーベルカール、ダンベルカール、プリチャーカール、ハンマーカール、ケーブルカール
- **上腕三頭筋**: トライセップスエクステンション、スカルクラッシャー、ナローベンチプレス、プッシュダウン、ケーブルエクステンション、ディップス
- **前腕**: リストカール、リバースリストカール

**腹筋（Abs）**
- **腹直筋・腹斜筋**: クランチ、シットアップ、レッグレイズ、プランク、サイドプランク、ロシアンツイスト、アブローラー

### 2. 部位別検索機能

**実装要件：**
```
- 部位をタップ→該当種目一覧表示
- 種目名での検索機能も並行実装
- 階層構造：大分類（脚）→小分類（大腿四頭筋）→種目（スクワット）
- 直感的なUI/UX設計
```

### 3. 重量選択UI改善

**実装仕様：**
```
- 1kg刻みの選択肢を下部に表示
- 範囲：1kg～200kg（必要に応じて拡張可能）
- スクロール可能な選択リスト
- 現在選択中の重量をハイライト
- ワンタップで選択完了
```

### 4. メール認証アカウント機能

**認証システム：**
```
- メールアドレス + パスワード認証
- アカウント作成・ログイン・ログアウト機能
- パスワードリセット機能
- セキュアなセッション管理
```

**個人情報管理：**
```json
{
  "user_account": {
    "email": "string",
    "password_hash": "string",
    "created_at": "timestamp"
  },
  "user_profile": {
    "height": "number",
    "weight": "number",
    "calibration_data": "object",
    "training_records": "array",
    "personal_bests": "object"
  }
}
```

### 5. データ統合・同期機能

**要件：**
- 既存のキャリブレーションデータをアカウントに紐づけ
- 全トレーニング記録の同期
- デバイス間でのデータ共有
- データバックアップ・復元機能
