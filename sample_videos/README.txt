サンプル動画ファイルについて
===================

このフォルダには、BodyScale Pose Analyzer用のサンプル動画を入れてください。
動画は以下の条件を満たしているとより良い結果が得られます：

1. MP4形式
2. 被験者の全身が映っていること
3. 安定したカメラアングル
4. 十分な明るさ
5. 背景と被験者の色のコントラストが良いこと

使用方法：
1. このフォルダに動画ファイルを配置
2. コマンドラインから以下のように実行:
   python batch_analyze.py 170 sample_videos/動画ファイル名.mp4
   (170は被験者の身長を示す値。実際の身長に合わせて変更してください)

注意：
* MediaPipeのポーズ検出が正確に動作するかどうかは動画の品質に依存します。
* 複雑な背景や照明条件では精度が低下する場合があります。
* 体の一部が隠れている場合、正確な測定ができないことがあります。
